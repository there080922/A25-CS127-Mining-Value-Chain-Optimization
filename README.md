Sistem ini dikembangkan untuk membantu proses pengambilan keputusan operasional pada industri pertambangan batubara. Operasional tambang dipengaruhi oleh berbagai faktor dinamis seperti:

- Cuaca

Proyek ini membangun **portal web** yang terintegrasi dengan **model AI** untuk mensimulasikan skenario operasional dan memberikan **rekomendasi keputusan** yang cepat, adaptif, dan mudah dipahami pengguna.
